# Installation

## Télécharger et lancer

1. Télécharge le dossier Sideviz depuis la page [Releases](../../releases/latest)
2. Place le dossier où tu veux sur ton PC
3. Ouvre le et double-clique sur le fichier .exe pour lancer l'application

Aucune installation requise, aucun droit administrateur. Un seul fichier suffit.

## Importer un profil de jeu

Un profil est un fichier json qui contient des réglages optimisés selon moi pour un jeu précis.

1. Télécharge le fichier json du profil voulu
2. Ouvre le dossier où se trouve Sideviz.exe
3. Ouvre le sous-dossier Profiles
4. Colle le fichier json dedans
5. Redémarre Sideviz

! Il vaut mieux éviter de mettre le même nom de fichier .json, car ça pourrait causer des problèmes.
Le profil apparaît ensuite dans la liste des profils dans les paramètres.


## Mettre à jour

Sideviz ne se met pas à jour tout seul : la mise à jour est manuelle, mais rapide.

1. Télécharge la dernière version sur la page [Releases](../../releases/latest).
2. Quitte Sideviz s'il est ouvert : clic droit sur l'icône dans la barre des tâches (en bas à droite) → **Quitter**.
3. Décompresse le fichier `.zip` téléchargé.
4. **Remplace uniquement le fichier `Sideviz.exe`** par le nouveau, dans ton dossier Sideviz existant.
5. Relance Sideviz.

> 💡 **Pourquoi remplacer seulement le `.exe` ?** Pour **garder tes profils**. Tes réglages et tes profils (y compris ceux que tu as créés) se trouvent dans le sous-dossier `Profiles`. En ne remplaçant que `Sideviz.exe`, tu ne risques pas de les perdre. Si tu préfères remplacer tout le dossier, pense d'abord à recopier tes fichiers `.json` personnels dans le nouveau `Profiles`.

Tu peux vérifier ta version dans les paramètres, onglet **À propos**.

## Désinstaller

Supprime le fichier Sideviz.exe et le dossier Profiles. Rien d'autre n'a été installé sur ton système.
